O mnie

Jestem technikiem IT i Java Developerem z kilkuletnim doświadczeniem w tworzeniu zaawansowanych systemów dla platform Minecraft — Paper, Spigot, Purpur, Velocity i BungeeCord.

Projektuję i wdrażam autorskie pluginy, systemy ekonomii, mechaniki rozgrywki, GUI, scoreboard i integracje z zewnętrznymi usługami. Na co dzień pracuję z MySQL/MariaDB, Redisem, Dockerem i administracją serwerami Linux.

Cenię czysty, testowalny kod, solidne architektury i dokumentację. Każdy projekt traktuję jako szansę na dostarczenie czegoś naprawdę wartościowego.

Java 21 Kotlin Paper API Velocity MySQL Redis Docker Linux Git

Oś Czasu

Każdy etap ukształtował moje podejście do tworzenia oprogramowania.

2019

Początek nauki programowania

Pierwsze kroki z Javą. Algorytmy, struktury danych, pierwsze projekty konsolowe. Zafascynowany tym, że komputery robią dokładnie to, co im powiesz.

2020

Pierwszy plugin Minecraft

Zbudowałem swój pierwszy plugin oparty na Spigot API — prosty system wiadomości prywatnych. Zaraziłem się na dobre.

2021

Pierwszy duży serwer Minecraft

Zaprojektowałem i wdrożyłem kompletną sieć serwerów Minecraft z proxy Velocity, własnym CorePlugin i systemami SkyBlock i PvP. Setki graczy dziennie.

2022

Technik IT

Poszerzyłem kompetencje o administrację Linux, Docker, sieci i bezpieczeństwo IT. Zdobyłem kwalifikacje technika informatyka.

2024 – teraz

Freelance & Open Source

Tworzę autorskie systemy na zlecenie, kontrybuuję do projektów open-source i rozwijam własne biblioteki. Specjalizacja: Paper API, Velocity, Redis i skalowalne architektury backendowe.
